# 블랙잭


## 기능 요구 사항

### 플레이어 이름

### 플레이어

- [ ] 플레이어는 이름을 갖는다
- [X] 플레이어는 카드들을 갖는다
- [X] 플레이어는 카드를 저장한다 
- [X] 죽었는지 안 죽었는지 여부를 반환한다 (21을 초과하는지)


### 카드

- [x] 문자와 모양을 상태로 갖는다 


### Hand(카드들)

- [x] 여러개의 카드를 갖는다
- [x] 카드의 총합 계산한다 
- [x] 카드 추가한다 


### 카드 게임

- [x] 각 플레이어에게 카드를 2장씩 지급한다
- [x] 플레이어마다 추가 지급한다


### 인풋 뷰

- [ ] 사용자 카드 합이 21을 초과하면, 카드 추가 여부를 묻지 않는다.
- [ ] 플레이어가 n을 입력할 때 까지 카드 추가 여부를 묻는다. 


### 아웃풋 뷰

- [ ] 딜러가 추가 카드를 발급 받았는지 여부 출력
